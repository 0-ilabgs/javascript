# JavaScript Learning Project

This project is a hands-on JavaScript practice workspace for learning the language. It contains small example files grouped by topic so you can explore variables, operators, arrays, console output, and comparisons.

## Project Structure

- `variables/` — examples using `const`, `let`, and variable declarations.
- `operators/` — basic math operations like addition, subtraction, multiplication, division, and modulus.
- `array/` — array practice code and simple array operations.
- `comparasionOpt/` — comparison and logical operator examples.
- `console/` — examples of `console.log`, `console.error`, `console.warn`, `console.table`, and similar console output.

## How to Use

1. Open the project folder in VS Code.
2. Review the files inside each folder to learn the topic.
3. Run an example file with Node.js from the terminal, for example:

```bash
node variables/var.js
node operators/addition.js
node array/array.js
node console/log.js
```

## Learning Notes

- Start with `variables/` to learn how JavaScript stores values.
- Then move to `operators/` to see how math works.
- Use `array/` to practice collections of values.
- Check `comparasionOpt/` to learn comparison logic and decisions.
- Use `console/` to see how output appears while running code.

## Requirements

- Node.js installed on your machine

## Tips

- Run one example at a time.
- Change values in the file and rerun to see how JavaScript behavior changes.
- Use the VS Code terminal for quick practice.
